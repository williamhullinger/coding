# CSS Flexbox Reference

Flexbox is a one-dimensional layout system for arranging items in rows or columns.

## Core Concepts
- Main axis vs cross axis
- Flex container and flex items

## Enable Flexbox
display: flex;

## Common Container Properties
- flex-direction
- justify-content
- align-items
- flex-wrap
- gap

## Common Item Properties
- flex
- flex-grow
- flex-shrink
- flex-basis
- align-self
- order

## Typical Use Cases
- Navigation bars
- Centering content
- Button groups
- One-dimensional layouts

## Mental Model
- Flexbox = layout along ONE axis (row OR column)
