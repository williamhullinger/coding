# CSS Grid Reference

CSS Grid is a two-dimensional layout system for rows AND columns.

## Core Concepts
- Grid container
- Grid items
- Tracks (rows / columns)
- Grid lines and areas

## Enable Grid
display: grid;

## Common Container Properties
- grid-template-columns
- grid-template-rows
- grid-template-areas
- gap
- justify-items
- align-items

## Common Item Properties
- grid-column
- grid-row
- grid-area

## Typical Use Cases
- Page layouts
- Dashboards
- Galleries
- Complex two-dimensional designs

## Mental Model
- Grid = layout across ROWS and COLUMNS
