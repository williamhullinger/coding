# Learning to Code — Web Development & Foundations

This repository documents my journey learning modern web development fundamentals.
It contains hands-on projects, reference notes, and supporting documentation as I
build real-world skills step by step.

The goal of this repo is **learning through building**, not just collecting tutorials.

## Projects

Each project folder represents a focused learning exercise or small application.
Every project contains its own files and a project-specific README explaining:

- What the project does
- What concepts are being practiced
- How to run or view the project

Projects are intentionally kept small and iterative so they can evolve as skills
and understanding improve.

---

## References

The `references/` folder contains personal reference notes and cheat sheets used
alongside project work.

These files are designed for quick lookup and reinforcement of core concepts,
not long-form tutorials.

Topics currently include:
- CSS units and layout systems
- HTML semantics and structure
- Git fundamentals and workflows
- Tooling and development environment notes

Reference files are living documents and may be updated as understanding deepens.

---

## Tech Focus

Current learning focus includes:
