# HTML Project Thinking Checklist

## 1. What is this page doing?
- Reading content?
- Entering information?
- Structured data?
- List of items?
- Page layout?

## 2. Choose the main container
- Form → user input
- Table → rows & columns
- ul / ol → lists
- header / main / section / footer → layout

## 3. Form checklist
- form element exists
- label for every input
- label for = input id
- required where needed

### Common inputs
- text
- email
- number (min/max)
- radio (same name)
- checkbox (value)
- select + option
- textarea
- submit button

## 4. Attribute check
- unique ids
- valid types
- correct min/max usage
- correct minlength/maxlength usage

## 5. Final question
Would this make sense to me in 3 months?
