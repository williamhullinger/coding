# HTML Semantic Elements Reference

Semantic elements describe the meaning and role of content.

## Why Semantics Matter
- Accessibility
- SEO
- Readability
- Maintainability

## Common Semantic Elements
- header
- nav
- main
- section
- article
- aside
- footer

## Example Structure
header
  nav
main
  section
    article
  aside
footer

## When to Use Semantic Elements
- When the element has a clear purpose
- When content has meaning, not just styling

## Rule of Thumb
If it has meaning → use semantic HTML
