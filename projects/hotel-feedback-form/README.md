# Hotel Feedback Survey Form

## Purpose
This project demonstrates how to collect user input using an HTML form.  
It was completed as part of a final exam and focuses on proper form structure,
accessibility, and semantic HTML usage.

## HTML Elements Used
- form
- fieldset
- legend
- label
- input
  - text
  - email
  - number
  - radio
  - checkbox
- select
- option
- textarea
- button
- header
- main

## Attributes Demonstrated
- action
- method
- type
- name
- id
- for
- placeholder
- required
- min / max
- checked
- selected

## Concepts Demonstrated
- Form submission using POST
- Grouping related inputs with fieldset and legend
- Associating labels with inputs for accessibility
- Required vs optional fields
- Radio buttons vs checkboxes
- Dropdown (select) menus
- Default selected values
- Semantic page structure (header + main)

## Notes
Good reference for:
- Surveys
- Feedback forms
- User input validation basics
- Accessibility-friendly form design

This project helped reinforce recognizing when a **form element** is the correct
root container for user input rather than tables or layout-based elements.
