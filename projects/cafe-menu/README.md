# Camper Cafe Menu ☕🍰

This project is my **first completed HTML + CSS project** and my **first time styling a webpage using CSS**.  
It was built as part of the **freeCodeCamp Responsive Web Design Certification**.

The goal of this project was to learn how **HTML structure** and **CSS styling** work together to create a clean, readable layout.

---

## Project Overview

The Camper Cafe Menu is a simple, responsive cafe menu that displays:
- A cafe title and establishment date
- Coffee and dessert sections
- Menu items aligned with names on the left and prices on the right
- A centered menu container over a background image
- Styled links with hover and active states

This project focuses on **CSS fundamentals** rather than advanced layouts or JavaScript.

---

## Files in This Project

- **`index.html`**
  - Contains the page structure and content
  - Uses semantic HTML elements such as:
    - `<main>`
    - `<section>`
    - `<article>`
    - `<footer>`
    - `<address>`
  - Organizes menu items into logical sections for readability and accessibility

- **`styles.css`**
  - Controls layout, colors, fonts, spacing, and alignment
  - Uses class selectors and grouped selectors
  - Demonstrates basic layout techniques using width percentages and margins
  - Styles link states for better user interaction

---

## Key Concepts Practiced

### HTML
- Semantic structure using sections and articles
- Proper use of headings and paragraphs
- Meaningful class names for styling (`menu`, `item`, `flavor`, `price`, `dessert`)
- Accessible image alt text
- External stylesheet linking

### CSS
- Background images and page padding
- Font families and font sizing
- Grouped selectors to reduce repetition
- Centering elements using auto margins
- Aligning content with `inline-block`
- Styling links using pseudo-classes (`:hover`, `:active`, `:visited`)
- Debugging common CSS issues (typos, comments, selector targeting)

---

## How the Layout Works (High Level)

- The menu content is wrapped in a centered container with a fixed maximum width
- Each menu item uses two text elements:
  - One for the item name
  - One for the price
- Width percentages allow item names and prices to align cleanly in a single row
- Typography and spacing create a menu-style layout that remains readable on different screen sizes
