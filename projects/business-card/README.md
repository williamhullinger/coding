# Business Card Project

This project is a simple digital business card built as part of the freeCodeCamp HTML & CSS curriculum.  
It demonstrates basic HTML structure, CSS styling, class selectors, and layout fundamentals.

---

## 📌 Project Overview

The business card includes:
- Profile image
- Name and designation
- Company name
- Contact information
- Portfolio link
- Social media links

The layout is centered on the page and styled to resemble a clean, readable digital business card.

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Visual Studio Code
- freeCodeCamp curriculum

---


## ✅ Completed Requirements

- Proper HTML document structure
- Semantic elements (`main`, `p`, `img`, `a`, `hr`)
- Class-based CSS styling
- Centered card layout
- Responsive image scaling
- Links styled without underlines (FCC requirement)

---

## 🎨 Current Styling Details

- Background color applied to the page
- Card container with padding and fixed width
- Centered alignment using margin auto
- Consistent typography
- Image scaled to container width

- Add `border-radius` to the card for softer edges

- Improve color contrast for accessibility

### Interaction & UX
- Add hover effects to links
- Add cursor styling for clickable elements

- Use Flexbox for tighter layout control

### Content Improvements
- Replace placeholder links with real URLs
- Add a short tagline or description
- Add icons for social media links

### Code Quality
- Group related CSS rules
- Use CSS variables for colors

- Normalize spacing and indentation
