# Blog Post Card

This project is a styled blog post card built with HTML and CSS. It focuses on layout fundamentals, card-style design patterns, and interactive link states using pseudo-classes.

---

##  Project Overview

The blog post card includes:
- A full-width cover image
- A post title
- A short excerpt
- A “Read more” call-to-action link

The layout follows common UI card patterns used in blogs, portfolios, and content feeds.

---

##  Concepts Practiced

### HTML
- Semantic structure using `<main>`
- Image elements with `src` and `alt`
- Grouping content with container elements
- Anchor elements with valid `href` values
- Class-based structure for styling

### CSS
- Card layout styling
- Fixed-width components
- Rounded corners using `border-radius`
- Image scaling with `width: 100%`
- Padding and margin for spacing
- Inline-block display for buttons
- Hover effects with pseudo-classes

---

##  Styling Features

- The card has a fixed width and rounded corners
- The image fills the entire card width and includes a bottom border
- Content spacing is handled with padding and margin
- The “Read more” link is styled as a button
- A hover effect provides visual feedback when interacting with the link
