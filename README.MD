# Learning to Code — Web Development & Foundations

This repository documents my journey learning modern web development fundamentals.
It contains hands-on projects, reference notes, and supporting documentation as I
build real-world skills step by step.

The goal of this repo is **learning through building**, not just collecting tutorials.

---

## Repository Structure

Projects are kept separate from reference material to keep everything organized and easy to review.


---

## Projects

Each project folder represents a focused learning exercise or mini-application.
Every project contains its own files and a project-specific README explaining:

- What the project does
- What concepts are being practiced
- How to run or view the project

Projects are intentionally kept small and iterative so they can evolve as skills improve.

---

## References

The `references/` folder contains personal technical notes and cheat sheets.
These files are written to reinforce understanding and serve as quick in-editor lookup guides.

Topics include:
- CSS units and layout systems
- HTML semantics
- Git fundamentals

These are living documents and may be updated as understanding deepens.

---

## Tech Focus

Current learning focus includes:
- HTML (semantic structure)
- CSS (layout, spacing, responsiveness)
- Git (version control basics)
- Development workflow using VS Code

JavaScript and more advanced topics will be added progressively.

---

## Purpose

This repository exists to:
- Practice real development habits
- Build a strong foundation before advanced frameworks
- Create a clear, reviewable learning trail

It is designed to grow over time as skills and projects become more advanced.
